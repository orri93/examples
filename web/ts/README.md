# TypeScript example

- Make sure that relevant version of [TypeScript](https://www.typescriptlang.org/download) is installed
- 
## Setup

- Create package.json file with only version and main
- Install packages

        npm install typescript --save-dev
        npm i lodash
        npm i -D @types/lodash

## Install

        npm install

## Compile

        npx tsc

## Start

        npm run

## Execute

        node src/index.js
